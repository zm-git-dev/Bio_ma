### 渗入区域的确定可以使用多种方法，一种方法为主，其他方法用来验证
如使用`rIBD`来确定渗入区域，可以使用`RFmix`和滑动窗口D以及FST等来证明此区域的准确性
```
算法越来越好了，目前推荐的软件有：

```
### 之后使用渗入区域位点及非渗入区域位点分别做系统发育树
若渗入区域系统发育树中目标群体有个体位于供体群体中，则证明此区域确实发生渗入
